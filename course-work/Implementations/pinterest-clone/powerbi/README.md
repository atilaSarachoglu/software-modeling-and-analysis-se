# Power BI Визуализации

## Инструкции за свързване

1. Отворете Power BI Desktop
2. Изберете "Get Data" → "SQL Server"
3. Въведете сървъра и базата данни (PinterestClone)
4. Импортирайте таблиците

## Предложени визуализации

### 1. Top Pins by Category
- **Тип**: Bar Chart
- **Данни**: Брой pins по категории
- **Филтри**: Период от време

### 2. User Engagement Over Time
- **Тип**: Line Chart
- **Данни**: Брой saves, comments, follows по дата
- **Филтри**: Тип активност

### 3. Board Growth Analysis
- **Тип**: Area Chart / Combo Chart
- **Данни**: Растеж на boards и pins във времето
